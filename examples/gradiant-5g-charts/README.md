# Gradiant 5G Charts Example

This example is dedicated to **[Gradiant 5G Charts](https://github.com/Gradiant/5g-charts)**.

## Instructions

Generate the Kubernetes architecture diagrams for Gradiant 5G Helm charts:
```sh
$ ./generate.sh
```
